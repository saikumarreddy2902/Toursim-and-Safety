### Common Issues:

1. **Python not found:**
   - Install Python 3.7+ from https://python.org
   - Add Python to your PATH

2. **Dependencies missing:**
   - Run the "Install Dependencies" option
   - Or manually: `pip install -r backend/requirements.txt`

   - Run the "Setup Database" option
   - Or manually: `cd backend && python -c "from app import init_db; init_db()"`

4. **Port 5000 in use:**
   - Change port in `backend/app.py`
   - Or kill existing process using port 5000
